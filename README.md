# Unity Project: Weapon & Animal Setup

## Описание проекта

Данный проект демонстрирует настройку импорта ассетов, создание материалов, префабов и аниматора для двух сущностей: оружия (CZ600) и животного (BlackTailedDeer). 
И самое важное - кастомный Автосборщик-интегратор для Unity, автоматизирующий настройку и сборку префабов оружия и животного из FBX-моделей, материалов и текстур.
## 📋 Требования к проекту

| Компонент | Версия |
|-----------|--------|
| Unity | 6000. LTS или новее |
| Render Pipeline | Universal Render Pipeline (URP) |
| Платформа | Windows |
